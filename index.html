Creación de la aplicación informativa para verificar código de descuento

Antes de todo, se debe crear una estructura de la aplicación.

Primero, introducir un encabezado. Por ejemplo: "Tu descuento aquí".

Luego, con la etiqueta 'input' y el atributo 'type="submit"' creamos un formulario para introducir el código que el cliente tiene para disfrutar del descuento. Con 'placeholder' indicamos "Escribe tu código". El código tendrá un patrón que revisará la cantidad de caracteres y números introducidos con 'maxlength="10"'. En caso de no cumplir, aparecerá el mensaje "Error".

Después de enviar el código con el botón "Enviar", la información se enviará al servidor para verificar los datos introducidos y, posteriormente, en una parte de la pantalla se mostrará la respuesta obtenida en función de los datos introducidos. En caso de ser aprobado, el mensaje podría ser, por ejemplo: "Su código está verificado y con este puede aprovechar un 5% de descuento en cualquier restaurante del centro comercial 'Los Arcos' en Sevilla. La promoción estará en vigor hasta el 20/10/2025". Este mensaje indica el tipo y la cantidad de descuento, el periodo de validez y el lugar de uso.

En caso contrario, si el código no está verificado o está caducado, aparecerá un mensaje correspondiente a la situación.
